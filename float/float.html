Pull me

The Holy Grail in web design is a responsive three column layout, that changes as the screen shrinks.

In this case, I've implemented it with the older CSS style of "float", "block" and "inline block". Essentially switching the display property of divs to inline when they go into a mobile screen, which means they wrap underneath each other in the classic mobile column view.

Why the tower blocks? I guess the "block" display put me in mind of high-rises, and "float" made me think of clouds.

Pull the screen in and hover over the buildings to see different CSS transition effects.
