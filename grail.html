Holy Grail, flexbox style

The so-called "Holy Grail" of web design is a two or three column layout with a menu and footer. The layout should expand and compress responsively when the screen shrinks or grows.

In this page I've implemented it with CSS Flexbox, a relatively modern technology that is not supported by all browsers. Arguably up to 95% of them when used with the correct prefixes.

In the CSS, I've split out the flexbox properties from the rest of the style and put them at the top, to make the basics you need really clear.

I have chosen to run with the Grail theme
