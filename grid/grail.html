Grid is great.

Barely 3 lines of code are needed to produce the 3 column and header layout that used to be known as the "Holy Grail of web development". Amazing.

display: grid;
grid-template-columns: 33%;
grid-gap: 10px;

Using grid-template-columns: none; with a media query for screen widths of under 400px is enough to flip this into a single column for the mobile view.

For this layout to work, you need to enable the flag in Chrome that allows the browser to use the experimental Grid technology. Paste chrome://flags into the URL bar and scroll down to the option to Enable experimental Web Platform features, and hit enable.
